# BMI-calculator

A simple and interactive BMI (Body Mass Index) Calculator built using React.js. This project demonstrates the use of React Hooks (useState), event handling, and conditional rendering to calculate and display BMI results based on user input.

A BMI (Body Mass Index) Calculator built using **React + Vite**.

## Features

- Calculate BMI instantly
- Clean UI
- Responsive design

## Tech Stack

- React
- Vite
- JavaScript
- CSS
